# AI Document Management Sample

## Overview
This repository manages rules and knowledge base for AI assistants (GitHub Copilot, Cline, Cursor). The management logic is located in the ai-docs directory.

For main information, please refer to the [ai-docs](./ai-docs/README.md) directory.

## Usage

How to introduce AI document management to your repository:

1. Copy the `ai-docs` directory to your repository
2. Copy the GitHub Actions workflow file from `./.github/workflows/check-ai-instructions.yml`
